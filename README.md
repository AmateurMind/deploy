# PIC Programs Viewer

A simple Streamlit app to view and copy PIC microcontroller program examples.

## Features

- View different PIC program examples
- Copy code directly to clipboard
- Download programs as .c files

## Deployment

1. Fork this repository
2. Go to [Streamlit Cloud](https://share.streamlit.io)
3. Sign in with GitHub
4. Deploy the app by selecting this repository and the main file `app.py`

## Local Development

```bash
pip install -r requirements.txt
streamlit run app.py
```
